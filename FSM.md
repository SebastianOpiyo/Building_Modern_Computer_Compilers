## Powers of Sigma(Σ) 
Σ^0 = set of all string of length 0: Σ^0 = {∈}
Σ^1 = set of all string of length 1: Σ^1 = {0, 1}
Σ^2 = set of all string of length 2: Σ^2 = {00, 01, 10, 11}
Σ^n = set of all string of length n: Σ^n
